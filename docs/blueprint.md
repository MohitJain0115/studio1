# **App Name**: FinanceFriend

## Core Features:

- Mortgage Calculator: Calculate monthly mortgage payments based on loan amount, interest rate, and loan term.
- Loan Calculator: Calculate loan payments and generate amortization schedules for various loan types.
- Savings Calculator: Estimate future savings based on initial investment, contribution frequency, and interest rate.
- Retirement Calculator: Project retirement income based on current savings, planned contributions, and estimated expenses. The LLM can act as a tool for analyzing the different expense amounts entered by the user and informing which scenarios they can reasonably consider based on present circumstances.
- Auto Loan Calculator: Determine the affordability of a vehicle purchase, including monthly payments, based on loan terms and interest rates.
- Investment Calculator: Evaluate investment growth and returns over time with customizable inputs.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) to convey trust and reliability.
- Background color: Light gray (#F0F0F0), a desaturated version of the primary color, for a clean, professional look.
- Accent color: Emerald green (#50C878) to indicate growth and financial success.
- Body and headline font: 'Inter', a sans-serif font, to offer a neutral and modern look.
- Use financial and calculation-related icons in a consistent, outline style.
- Employ a clean, grid-based layout with clear sections for each calculator. Ensure responsiveness across devices.
- Incorporate subtle transitions and animations for user interactions, such as loading results or toggling calculator options.
